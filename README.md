# TaskFlow
A full-stack task management app with auth, CRUD, and deployment

Tech stack:

- Backend: FastAPI
- Frontend: React
- DB: PostgreSQL
- Deployment: Render, Railway, or AWS
  
# Week 1 Backend Structure

Build:
- Project skeleton
- Database connection
- User model
- Register/login endpoints
- JWT authentication
- CRUD for tasks (create, read, update, delete)

Deliverables:
- FastAPI
- PostgreSQL + migrations
- /auth/register, /auth/login
- /tasks CRUD
- Swagger/docs auto-generated

# Week 2

Build: Frontend UI (React)
- Login + Register UI
- Dashboard showing tasks
- Create Task modal
- Edit/Delete buttons
- State management (React Query or Redux)
- Theme + clean UI (Material UI)

Deliverables:
- Full login/logout flow
- Fetch tasks from backend
- UI for adding/updating tasks
- Protected routes

# WEEK 3: Polish + Deployment

Build:
- Backend deployment → Render/Railway
- Frontend deploy → Netlify/Vercel
- Secure env variables
- Connect frontend to hosted backend
- Add pagination + filtering
- Add loading states + error toasts

Deliverables:
- Live website
- Live API
- GitHub repo with README
- Screenshots/gifs

# WEEK 4: Stretch Goals

Add:
- Logging
- Rate limiting
- Error monitoring (Sentry)
- Dockerfile
- CI/CD (GitHub Actions)

Deliverables:
- Demonstrates backend + frontend + deployment
